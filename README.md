# WCAG Accessibility Audit (Bachelor Thesis)

This project is my bachelor thesis in Frontend Development, focusing on
accessibility and user experience through WCAG 2.1 Level AA compliance.

## Scope
- Accessibility evaluation of three Swedish public-facing websites
- Based on WCAG 2.1 AA
- Tools used: Lighthouse and Axe
- Manual analysis supported automated testing

## Websites analyzed
- Sportamore.se
- Willys.se
- Apoteket.se

## Focus areas
- Perceivable, Operable, Understandable, Robust (POUR)
- Contrast, keyboard navigation, semantic structure
- Accessibility impact on overall user experience

## Key takeaways
- Automated tools are helpful but insufficient alone
- Many accessibility issues stem from design decisions
- Accessibility improvements often improve UX for all users
